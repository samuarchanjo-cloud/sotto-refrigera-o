# Sotto Refrigeração e Climatização

Landing page institucional em React, Vite, TypeScript, Tailwind CSS e Framer Motion para a Sotto Refrigeração e Climatização.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```

## Conectar ao GitHub e publicar

```bash
git remote add origin URL_DO_REPOSITORIO
git branch -M main
git push -u origin main
vercel --prod
```

O projeto já inclui `vercel.json` com rewrite para SPA em React + Vite.
